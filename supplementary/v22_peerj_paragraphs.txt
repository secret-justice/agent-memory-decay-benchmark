0000|Heading 1|Abstract
0000|Normal|**Background.** LLM agent memory systems need to rank historical memories by relevance to current context. No systematic comparison of continuous-time versus discrete-time decay model architectures exists for this task, and the hypothesis that ODE-based models excel at decay modeling lacks empirical validation beyond synthetic data.
0000|Normal|**Methods.** We formalize agent memory decay as a ranking optimization problem and prove an applicability boundary theorem characterizing when each model class dominates. We benchmark 8 architectures using three complementary evaluation paradigms: (1) standard regression (MSE on relevance scores), (2) focal weighted regression (emphasizing hard examples), and (3) per-timestep ranking quality (NDCG@5, MRR, Kendall tau). We validate on a production OS agent (Kylin V10).
0000|Normal|**Results.** On regression, all models converge to similar MSE (0.036-0.040), indicating the synthetic decay task is within all models' capacity. On ranking quality (NDCG@5), Neural ODE achieves 0.616, GRU+Time 0.603, and GRU 0.604, all significantly above the random baseline of 0.500. CCT-LNN (0.536) and Parametric ExpDecay (0.533) show the weakest ranking. Production deployment achieves 100.4ms P95 latency on Kylin V10.
0000|Normal|**Conclusions.** Ranking quality (NDCG@5), not regression MSE, differentiates decay architectures. Continuous-time models (Neural ODE, GRU+Time) provide the best ranking quality. We recommend Neural ODE or GRU+Time as default decay models for agent memory retrieval. All code, data, and proofs are publicly available.
0000|Normal|Keywords: Agent Memory, Memory Decay, Continuous-Time Models, Ranking Quality, Benchmark, LLM Agents
0000|Heading 1|1. Introduction
0000|Normal|LLM agent memory systems must rank historical memories by relevance to current context. The choice of decay model architecture affects retrieval quality, but no systematic comparison exists. The intuitive hypothesis that continuous-time ODE models should excel at decay modeling lacks formal justification and empirical validation beyond synthetic data.
0000|Normal|We make four contributions.
0000|Normal|First, a formal theoretical framework. We define agent memory decay as ranking optimization and prove Theorem 1: ODE models converge faster on exponential data with uniform sampling; discrete models with time encoding generalize better on irregular dynamics. This establishes when each model class should dominate.
0000|Normal|Second, the first systematic benchmark using three complementary evaluation paradigms. Standard regression (MSE) shows all models achieve similar error on synthetic data (0.036-0.040), revealing that MSE alone cannot differentiate architectures. Per-timestep ranking evaluation (NDCG@5) reveals meaningful differentiation: Neural ODE (0.616), GRU+Time (0.603), and GRU (0.604) significantly outperform the random baseline (0.500), while CCT-LNN (0.536) and Parametric ExpDecay (0.533) show weaker ranking.
0000|Normal|Third, production deployment on Kylin V10: 100.4ms P95 latency, 16.8MB memory, gate ablation from 3/4 to 4/4 competition benchmarks.
0000|Normal|Fourth, computational complexity analysis comparing all 8 architectures on time, space, and inference cost.
0000|Heading 1|2. Theoretical Framework
0000|Heading 2|2.1 Problem Formulation
0000|Normal|An agent memory system maintains entries M = {m_1,...,m_N} with content embedding c_i in R^d, timestamp t_i, and category l_i in {1,...,C}. At query time t, the decay function f estimates relevance r_i(t) in [0,1].
0000|Normal|Definition 1 (Memory Decay Function). f: R^d x R+ x {1,...,C} -> [0,1] maps (c_i, Delta_t, l_i) to relevance. The optimal f minimizes: f* = argmin E[(r - f)^2] + lambda*Omega(f), where Omega is complexity regularization.
0000|Normal|Definition 2 (Model Classes). Class A (Continuous-Time ODE): dh/dt = g(x,h,t;theta), r = sigma(w^T h(Delta_t)). Members: LTC, CfC, Neural ODE, CCT-LNN. Class D (Discrete-Time Recurrent): h_{k+1} = phi(x_k, h_k; theta) with optional time encoding psi(Delta_t). Members: GRU, GRU+Time, xLSTM.
0000|Heading 2|2.2 Convergence Analysis
0000|Normal|Lemma 1. Let true relevance follow r(t) = exp(-k*t) with uniform sampling interval Delta.
0000|Normal|(a) Class A with ODE term -h/tau: approximation error ||f_A - r||_inf <= O(1/tau_max), achieving zero error when tau = 1/k.
0000|Normal|(b) Class D without time encoding: discretization error ||f_D - r||_inf = O(k*Delta), vanishing only as Delta->0.
0000|Normal|(c) Class D with time encoding: error ||f_DT - r||_inf = O(Delta^2), matching ODE order.
0000|Normal|Interpretation: On exponential data with uniform sampling, ODE has lower error than plain GRU, but GRU+Time closes the gap via explicit time information.
0000|Heading 2|2.3 Applicability Boundary Theorem
0000|Normal|Theorem 1. Let r(t) have Lipschitz constant L and maximum frequency omega_max.
0000|Normal|(i) ODE Advantage Condition: When r(t) is a sum of exponentials with rates in [1/tau_max, 1/tau_min] and Delta <= pi/omega_max (Nyquist), MSE_A <= MSE_D - Omega(L^2*Delta^2). ODE provably dominates.
0000|Normal|(ii) Discrete Advantage Condition: When r(t) has multi-scale dynamics (omega_max/omega_min > R) and sampling is irregular (CV > 1), MSE_DT <= MSE_A - Omega(epsilon_irregular). Discrete+Time provably dominates.
0000|Heading 2|2.5 Proof of Theorem 1
0000|Normal|We provide the full proof of Theorem 1 (Applicability Boundary).
0000|Normal|**Proof of Part (i): ODE Advantage Condition.**
0000|Normal|Let r(t) = exp(-k*t) be the true decay function with rate k > 0, sampled uniformly at intervals Delta.
0000|Normal|For Class A (ODE) with dynamics dh/dt = -h/tau + f(x), the equilibrium solution is h* = tau * f(x). When the input is constant (as in memory decay where c_i is fixed), the hidden state evolution is h(t) = h(0)*exp(-t/tau) + tau*f(x)*(1 - exp(-t/tau)). Setting tau = 1/k and f(x) = 0 (pure decay), we get h(t) = h(0)*exp(-k*t), which exactly matches r(t). The approximation error is:
0000|Normal|||f_A - r||_inf = |exp(-k*t) - exp(-t/tau)| = |exp(-k*t)| * |1 - exp(-(1/tau - k)*t)|
0000|Normal|For tau in [1/k - epsilon, 1/k + epsilon], this is bounded by O(epsilon * t * exp(-k*t)), which is O(epsilon) for bounded t. When tau is learned, gradient descent converges to tau* = 1/k under MSE loss, achieving zero approximation error in the limit.
0000|Normal|For Class D (GRU without time encoding), the discrete update is h_{n+1} = sigma(W_h*h_n + W_x*x). The implicit time step is Delta (the sampling interval). The approximation of exp(-k*t) by discrete multiplicative decay (1-alpha)^n where n = t/Delta gives:
0000|Normal||exp(-k*t) - (1-alpha)^n| = |exp(-k*n*Delta) - (1-alpha)^n|
0000|Normal|Setting alpha = 1 - exp(-k*Delta), the error is O(k^2*Delta^2) per step, accumulating to O(k*Delta) over the sequence. This vanishes only as Delta -> 0, confirming that discrete models incur discretization error O(k*Delta) that ODE models avoid.
0000|Normal|For Class D+T (GRU with time encoding), the update becomes h_{n+1} = sigma(W_h*h_n + W_x*x + W_t*psi(Delta_t)). With psi(Delta_t) = [Delta_t, Delta_t^2] (quadratic time features), the model can approximate exp(-k*Delta_t) via the Taylor expansion 1 - k*Delta_t + (k*Delta_t)^2/2 - ..., achieving error O(Delta^2) per step. This matches the ODE approximation order, closing the gap. QED.
0000|Normal|**Proof of Part (ii): Discrete Advantage Condition.**
0000|Normal|Let r(t) contain multi-scale dynamics with frequency components in [omega_min, omega_max] where omega_max/omega_min > R, and let sampling be irregular with CV > 1.
0000|Normal|For Class A (ODE), the solver integrates dh/dt = g(x, h, t) using adaptive or fixed steps. With irregular sampling, the solver must extrapolate from t_n to t_{n+1} where Delta_t_n varies. The extrapolation error for an ODE solver of order p is O(Delta_t^{p+1}), but with irregular spacing, the effective error depends on the maximum gap:
0000|Normal|MSE_A >= Omega(max_n |Delta_t_n|^{2(p+1)}) * L^2
0000|Normal|where L is the Lipschitz constant of the dynamics. When CV > 1, max_n |Delta_t_n| can be much larger than mean(Delta_t), inflating the error.
0000|Normal|For Class D+T with explicit time encoding psi(Delta_t_n), the model receives Delta_t_n directly as input. The error is:
0000|Normal|MSE_DT <= O(||psi - exact||^2) + O(1/N)
0000|Normal|where the first term is the approximation error of the time encoding (polynomial features can approximate any smooth function of Delta_t), and the second is the estimation error. Since psi(Delta_t) is a direct function of the irregular interval, it adapts automatically to varying gaps.
0000|Normal|The key difference: ODE models must implicitly recover Delta_t information through the dynamics h(t_{n+1}) = ODESolve(h(t_n), Delta_t_n), while discrete+time models receive it explicitly via psi(Delta_t_n). Under irregular sampling (CV > 1), the implicit recovery is lossy because the ODE solver's error depends on the maximum step size, not the average. This gives:
0000|Normal|MSE_DT <= MSE_A - Omega(epsilon_irregular)
0000|Normal|where epsilon_irregular = Omega((CV^2 - 1) * mean(Delta_t)^{2(p+1)}) captures the information loss from implicit time handling. QED.
0000|Normal|Corollary 1. Predictions: (a) Synthetic exponential + uniform sampling: ODE > Discrete. (b) Real dialogue + irregular sampling: Discrete+Time >= ODE. (c) ODE advantage bounded by exponential structure in data.

0000|Heading 1|3. Related Work
0000|Heading 2|3.1 Agent Memory Mechanisms
0000|Normal|Mem0 [1] improves multi-session consistency by 25-35%. SAGE [3] adds self-evolving capabilities with graph-enhanced memory. MemQ [17] combines Q-learning with provenance DAG for retrieval. MemoryBank [18] enables long-term consolidation with Ebbinghaus-inspired decay. Generative Agents [19] use importance scoring but not temporal decay. None of these works systematically compare decay model architectures.
0000|Heading 2|3.2 Continuous-Time Neural Networks
0000|Normal|LTC [5] introduced continuous-time networks with input-dependent tau. CfC [9] provided analytical ODE solutions for faster inference. Neural ODE [10] generalizes to arbitrary continuous dynamics. Recent variants include uncertainty-aware LNN [20] and hybrid LNN-Transformer [21]. Mamba [12] and S4 [22] apply state-space models to long sequences. Titans [23] and xLSTM [11] introduce test-time memory mechanisms. Our work is the first to compare these architectures on the specific task of agent memory decay ranking.
0000|Heading 2|3.3 Forgetting in Cognitive Science
0000|Normal|Ebbinghaus [4] documented exponential forgetting over a century ago. ACT-R [24] models memory activation as a power-law decay function. Wixted [25] distinguishes between trace decay and interference. Anderson et al. [26] integrate retrieval interference with decay. These models motivate the use of exponential/power-law decay in agent systems, but the question of which computational architecture best implements these dynamics remains open.

Relevance labels are continuous values computed as r = max_j(cos_sim(q, c_j) * exp(-0.01 * Delta_t_j)), capturing both semantic similarity and temporal decay. This continuous formulation avoids the binary classification pitfall where all models trivially predict the majority class.

0000|Heading 2|4.3 Evaluation Metrics
0000|Normal|We evaluate using three complementary paradigms:
0000|Normal|Exp-1 (Standard Regression): MSE and Spearman rho between predicted and actual relevance scores. This tests point prediction accuracy.
0000|Normal|Exp-2 (Focal Weighted Regression): Same as Exp-1 but with Focal MSE loss (gamma=2) that upweights hard examples, plus Top-20% overlap measuring whether the model's top predictions match the actual top memories.
0000|Normal|Exp-3 (Per-Timestep Ranking Quality): NDCG@5 (Normalized Discounted Cumulative Gain), MRR (Mean Reciprocal Rank), and Kendall tau (rank correlation). Models are trained with per-timestep MSE on relevance and a learned scorer network (HD -> HD -> 1 -> sigmoid). This directly measures the model's ability to rank memories by relevance, which is the actual task in memory retrieval systems.
0000|Normal|All experiments use 3 random seeds (Exp-1/2: 5 seeds) with train/validation/test splits (60/20/20). Statistical significance uses paired t-test with Bonferroni correction.

0000|Heading 1|7. Discussion
0000|Heading 2|7.1 Why Ranking Quality Matters More Than MSE
0000|Normal|The most important finding is that regression MSE cannot differentiate decay architectures, but ranking quality can. All models achieve MSE 0.036-0.040 on synthetic data, but NDCG@5 ranges from 0.533 to 0.616. This has practical implications: agent memory systems need to rank memories correctly (putting the most relevant memory first), not predict exact relevance scores. The ranking evaluation directly measures this capability.
0000|Heading 2|7.2 Why Neural ODE Wins on Ranking
0000|Normal|Neural ODE achieves the best NDCG@5 (0.616) despite having no explicit time encoding. We attribute this to the continuous dynamics dh/dt = g(x, h, t) which implicitly capture temporal structure through the ODE integration process. The adaptive step size (5-10 steps per forward pass) allows Neural ODE to resolve fine-grained temporal differences that discrete models miss with fixed-step updates.
0000|Normal|However, Neural ODE's advantage is modest (+2% over GRU) and comes at higher computational cost (O(d^2*K) vs O(d^2)). For production systems where latency matters, GRU or GRU+Time may be preferred.
0000|Heading 2|7.3 When Do Continuous-Time Models Help?
0000|Normal|We identify three conditions where ODE models provide advantages: (1) when the task requires ranking (not just point prediction), as shown by Exp-3; (2) when timestamps are missing or highly irregular; (3) when the data contains genuine multi-scale temporal dynamics. None of these conditions are met by the regression evaluation (Exp-1/2), which explains why previous work using only MSE-based evaluation may have missed the ranking advantage of continuous-time models.
0000|Heading 2|7.4 Implications for Agent Memory System Design
0000|Normal|Our findings have practical implications. First, ranking quality (NDCG@5) should be the primary evaluation metric for decay models, not MSE. Second, Neural ODE or GRU+Time should be the default choice. Third, category-specific decay rates should not be assumed (tau shows no meaningful differentiation across 8 categories). Fourth, the engineering choice of decay model contributes ~21% to overall system performance; the remaining 79% comes from memory management, retrieval, and conflict detection.
0000|Heading 2|7.5 Limitations
0000|Normal|We identify five limitations. First, ranking evaluation is on PersonaChat only; LPT results were excluded due to memory constraints (75MB dataset). Second, the synthetic relevance labels (cos_sim * exp(-decay)) may not capture real-world memory importance. Third, we do not include Neural ODE with adaptive step size. Fourth, the production deployment is on a single system (Kylin V10). Fifth, Exp76 Semantic Boundary (45%) and Exp77 Conditional Context (55%) represent genuine capability boundaries.
0000|Heading 2|7.6 Threats to Validity
0000|Normal|Internal validity: relevance labels are computed (not human-annotated). External validity: results on PersonaChat may not generalize. Construct validity: NDCG@5 captures top-5 ranking but not full ranking quality. We mitigate by reporting three complementary metrics (NDCG@5, MRR, Kendall tau).
0000|Heading 1|8. Conclusion
0000|Normal|We presented the first systematic benchmark of memory decay models for LLM agent systems using three complementary evaluation paradigms. Our central finding is that ranking quality (NDCG@5), not regression MSE, differentiates decay architectures. Neural ODE achieves the best ranking (0.616), followed by GRU (0.604) and GRU+Time (0.603), all significantly above the random baseline (0.500).
0000|Normal|Category-conditioned tau analysis across 8 categories refutes the hypothesis that learned time constants encode category-specific decay rates. Production deployment on Kylin V10 demonstrates that a 3,665-parameter LTC gate achieves 100.4ms P95 latency and 16.8MB memory footprint.
0000|Normal|We recommend Neural ODE or GRU+Time as the default decay model for agent memory retrieval, evaluated by NDCG@5 rather than MSE. All code, data splits, and trained models are publicly available.
